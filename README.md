# Travel Recommendation Web System

A web-based travel recommendation system that provides personalized destination suggestions using **content-based filtering**.  
This system focuses on **5 major cities in Indonesia** and analyzes tourist reviews, ratings, and destination attributes to generate relevant travel recommendations.

## Project Overview

This project aims to help users discover suitable travel destinations based on their preferences.  
By analyzing destination characteristics and user-related data, the system recommends places that match user interests efficiently.

**Supported Cities:**
- Yogyakarta  
- Bali  
- Jakarta  
- Bandung  
- Malang

## Features

- Content-based filtering recommendation algorithm  
- Analysis of tourist reviews and destination attributes  
- User-friendly web interface  
- Dynamic city-based destination recommendations  

## Technology Stack

- **Backend:** Python (Flask)  
- **Frontend:** HTML, CSS, JavaScript  
- **Recommendation Method:** Content-Based Filtering  
- **Data Processing:** Tourist reviews & destination attributes
